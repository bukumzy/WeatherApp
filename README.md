# WeatherApp
React-based weather dashboard with real-time data from OpenWeatherMap.

## Features
- Search weather by city name
- Display temperature, humidity, and wind speed
- Weather condition icons
- Responsive design for desktop and mobile
- Error handling for invalid city searches
- Real-time weather updates

## Technologies Used
- React JS
- Tailwind CSS
- OpenWeatherMap API
- JavaScript

## Project Status
This project is currently under development as part of a frontend capstone project.

## Deployment
The application will be deployed once development is complete.
